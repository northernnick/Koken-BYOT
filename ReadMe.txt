/* Introduction
----------------------------------------------- */

The four themes included in this archive are derived from a four-part series of articles that explain how to build your first Koken theme. Check out the first article in that series here:

http://help.koken.me/customer/portal/articles/828691-build-your-first-theme

/* Installation
----------------------------------------------- */

Using your FTP client, navigate to /koken/storage/themes/. Drag any/all of the the "BYOT" folders to that directory. Refresh the Koken console, then go to Site > Themes to preview.
